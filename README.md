# Вёрстка таблицы

Реализовать html-страницу с выбором плана подписки.

Требования:
- реализовать с помощью таблицы;
- при задании стилей не создавайте ваши классы, а используйте псевдоклассы `first-child`, `last-child`, `nth-child`.
